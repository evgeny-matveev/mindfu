Overview

Terminal user interface for meditation player Displays current file and playback controls

This class provides a text-based user interface using the curses library. It displays the current playing file, playback state, and keyboard controls for user interaction.

def draw_controls
  controls = [
    "[SPACE] Play/Pause",
    "[S] Stop",
    "[N] Next",
    "[P] Previous",
    "[Q] Quit"
  ]

  @window.setpos(7, 2)
  controls.each_with_index do |control, i|
    @window.addstr(control)
    @window.setpos(7 + i, 2) if i < controls.length - 1
  end
end

def draw_status
  filename = @state.current_filename || "No file"
  state = @state.state.to_s.upcase

  @window.setpos(2, 2)
  @window.addstr("File: #{filename}")
  @window.setpos(3, 2)
  @window.addstr("State: #{state}")
end

Handle keyboard input from user

Processes keyboard input and triggers appropriate state machine events: SPACE - Play/Pause toggle S - Stop playback N - Next track P - Previous track Q - Quit application

Supports both English and Russian keyboard layouts: English: SPACE, S, N, P, Q Russian: SPACE, Ы, Т, З, Й

def handle_input
  case @window.getch
  when " "
    if @state.playing?
      @state.pause
    elsif @state.paused?
      @state.resume
    else
      @state.play
    end
  when "s", "S", "ы", "Ы"
    @state.stop
  when "n", "N", "т", "Т"
    @state.next
  when "p", "P", "з", "З"
    @state.previous
  when "q", "Q", "й", "Й"
    @running = false
  end
end

def init_curses
  Curses.init_screen
  Curses.curs_set(0)
  Curses.noecho
  Curses.stdscr.keypad(true)
  @window = Curses.stdscr
end

def run
  init_curses
  @running = true

  while @running
    draw
    handle_input
    sleep(0.1)
  end

  close_curses
rescue StandardError => e
  close_curses
  raise e
end
